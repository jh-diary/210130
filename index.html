        입력 태그#2
        

          
textarea
 
 여러줄의 텍스트를 입력받을 때 사용하는 태그이며 영역의 크기는 rows와 cols로 설정

          
select
 
메뉴를 표시하고 사용자로 하여금 선택하게 하는 요소이다. select태그는 메뉴리스트를 나열할 때 option태그를 같이 사용하며 selected속성을 통해 특정 항목을 초기에 설정이 가능하다.

          
fieldset
 
입력요소를 그룹핑하는 태그이며 legend태그를 사용하면 그룹에 제목을 붙일 수 있다.

          
label
 
 input 요소를 위한 범위를 정의한다. 예를 들어 글씨를 클릭해도 버튼에 표시가되는 것처럼 마우스 사용자에게 편리하게 클릭의 범위를 넓히는 것!

          
input type="file"
 
사용자가 서버로 파일을 전송할 때 이 태그를 사용하며 이 태그는 속성으로 enctype="multi-part/form-data"를 추가해주는 것이 좋다. 또한 input 태그의 accept 속성은 업로드시킬 파일의 MINE 타입을 알려준다 EX) img/gif img/jpg img/png

          
input type="hidden"
 
클라이언트 컴퓨터가 서버 컴퓨터로 특정 데이터를 전송하고싶을 때 사용하고 이 때 input type="hidden"의 태그의 name과 value 속성값이 전송된다.

        

      

      

        HTML 5 입력 요소
        date,datetime,datetime-local,month,time,week,color,email,tel,search,range,number,url의 input type=""가 많이 생겼다.
        추가 속성으로는 autocomplete,autofocus,placeholder,readonly,required,pattern 등이 생겼다.
        

          
input type="search"
 
 사이트 안에서의 검색이나 블로그 검색을 포함하여 모든 검색의 입력양식은 search로 표기한다.

          
input type="email"
 
이메일이 아닌지 맞는지에 대한 검사가 포함되어있다. 유효한 주소 검사는 "제출" 버튼을 눌렀을 때 사용가능한다. 정교한 검사를 하려면 pattern속성을 지정하면된다. patter속성에는  정규식을 넣어준다
          정규식은 특정한 규칙을 가지고 문자열을 표현하는 수식이며 정규식은 검색과 치환에 사용되지만 이는 좀 더 나중에 공부하는 걸로 하자. 다 외우기 힘들다.

          
input type="url"
 
이도 email과 같이 검사가 포함되어있는데 검증 작업을 시키려면 required 속성을 설정하자.

          
input type="tel"
  
 이것도 비슷한데 검증을 올바르게 하려면 pattern 속성 실행

          
input type="number"
 
 max,min,step 속성을 이용하여 최댓값,최솟값,단계값을 지정가능하다.

          
input type="range"
 
 number와 같지만 step이 아닌 value값을 사용한다.

          
input type="날짜의 관한 모든 값"
 
 min,max 속성 지정 가능
